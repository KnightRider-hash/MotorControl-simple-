# MotorControl-simple-
Mini motor speed control using hc05 paired with arduino
## Component required
-Arduino 
-HC-05
-Resistors(1k ohm & 2k ohm)
-Motor
-Mosfet(i used N channel mosfet)
-Battery
-Wires
## Working 
Paired hc05 with phone using bluetooth.When values (1,2,3,4) are pressed motor starts spinning with different speed ,example if pressed 1-it represent lowest speed and 4-represents highest speed.There is also a 0 value for turniung the motor off.
